I work as a Data Analyst at an innovative start-up in Leipzig, Germany. I graduated from Hochschule Mittweida, University of Applied Sciences Germany, with a degree in "Applied Mathematics". My master thesis *"Deep learning based classification of liquids using ultrasound sensor system data"* talks about applying deep learning to classify liquids. 

I started this blog partly because it would help me improve my writing skills and largely since i kept having lot of random ideas in my head and new learnings each day and i wanted to write them down somewhere. 

I grew up in Mumbai, India where i had completed my undergrad in Statistics. Back in India i was largely involved in Finance and Investment banking. 

When I write code I usually do it in Python which is more of my go-to language for Deep learning. I love to visualise data and statistically analyse them, for which i use R. Most of what I’ve been doing last few years has been machine learning (usually Deep learning), Statistical Analysis.
